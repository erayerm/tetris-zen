# Zen Tetris

An endless, calm, cozy Tetris inspired by tetr.io's Zen mode.

**▶️ [Play in your browser](https://erayerm.github.io/tetris-zen/)** · [Windows download](https://github.com/erayerm/tetris-zen/releases/latest)

**Controls:** ←/→ move · ↓ soft drop · Space hard drop · ↑/X rotate · Z rotate CCW · C/Shift hold · Esc pause
